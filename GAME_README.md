# 数数乐 - 微信小游戏

## 游戏简介

这是一款基于微信小游戏平台开发的数字点击游戏，玩家需要按照数字从小到大的顺序依次点击不规则图形。

## 游戏特性

### 核心功能
- **随机图形生成**：通过随机直线切割矩形区域，生成多种不规则多边形
- **智能数字显示**：数字大小根据图形面积自适应调整，确保清晰可辨
- **精确点击检测**：使用射线投射算法实现精确的点在多边形内检测
- **视觉反馈系统**：点击正确时高亮显示，点击错误时震动效果

### 游戏机制
- **顺序点击**：玩家必须按照1到N的顺序点击图形
- **进度追踪**：实时显示当前点击数字和总体进度
- **通关判定**：按正确顺序点击完所有数字即通关
- **时间统计**：记录完成游戏所用时间

### 难度系统
- **简单模式**：5个图形，切割次数较少
- **普通模式**：10个图形，中等切割复杂度
- **困难模式**：15个图形，高切割复杂度

### 用户体验
- **简洁界面**：包含开始按钮、重置按钮和游戏说明
- **音效系统**：点击音效、错误音效、通关音效
- **进度保存**：自动保存最佳成绩记录
- **响应式设计**：适配不同屏幕尺寸

## 技术架构

### 核心类结构

#### Polygon (多边形类)
- 管理多边形顶点、颜色和数字标识
- 实现点在多边形内检测算法
- 处理高亮、缩放、震动等视觉效果

#### PolygonGenerator (图形生成器)
- 实现随机直线切割算法
- 生成多种不规则多边形形状
- 支持不同难度级别的图形生成

#### GameManager (游戏管理器)
- 管理游戏状态和逻辑
- 处理点击事件和游戏流程
- 实现通关判断和时间统计

#### UI (用户界面)
- 实现菜单、游戏界面和通关界面
- 处理按钮交互和界面切换
- 显示游戏进度和说明信息

#### SoundManager (音效管理器)
- 管理游戏音效播放
- 支持音量控制和开关
- 处理背景音乐播放

#### FindGameMain (游戏主类)
- 协调各个模块的运行
- 处理游戏主循环
- 管理游戏进度保存和加载

### 关键算法

#### 图形切割算法
1. 从矩形区域开始
2. 生成随机水平或垂直切割线
3. 使用线段相交算法计算切割点
4. 将多边形分割为两个新的多边形
5. 重复切割直到达到目标数量

#### 点击检测算法
使用射线投射算法（Ray Casting）：
1. 从点击点向任意方向发射射线
2. 计算射线与多边形边的交点数量
3. 交点数量为奇数则点在多边形内
4. 交点数量为偶数则点在多边形外

#### 自适应字体大小
根据多边形面积计算字体大小：
```javascript
fontSize = Math.max(16, Math.min(32, Math.sqrt(area) / 3))
```

## 使用说明

### 开发环境
1. 安装微信开发者工具
2. 导入项目目录
3. 选择小游戏项目类型
4. 编译运行

### 游戏操作
1. 点击"开始游戏"按钮开始
2. 点击"难度"按钮切换难度级别
3. 点击"游戏说明"查看游戏规则
4. 按照数字顺序点击图形
5. 点击"重新开始"重置当前游戏
6. 点击"返回菜单"回到主菜单

### 音效文件
游戏需要以下音效文件（需要自行添加）：
- `audio/click.mp3` - 点击正确时的音效
- `audio/error.mp3` - 点击错误时的音效
- `audio/complete.mp3` - 通关时的音效
- `audio/bgm.mp3` - 背景音乐（可选）
