# 关卡系统更新说明

## 更新概述

已成功将游戏从难度选择系统改为关卡系统，现在游戏包含两个关卡：
- **第一关**：10个图形
- **第二关**：100个图形

## 主要改动

### 1. UI界面改动

#### 移除的功能
- ❌ 难度选择按钮（简单/普通/困难）
- ❌ 难度切换功能

#### 新增的功能
- ✅ 关卡选择按钮（关卡: 1/2）
- ✅ 关卡切换功能
- ✅ 通关后显示"下一关"按钮（仅在有下一关时显示）

### 2. 关卡配置

```javascript
levelConfig = {
  1: { count: 10, name: '第一关' },
  2: { count: 100, name: '第二关' }
}
```

### 3. 菜单界面更新

**主菜单按钮布局**：
1. **开始第一关** / **开始第二关** - 根据当前关卡显示
2. **关卡: 1/2** - 点击切换关卡
3. **游戏说明** - 查看游戏规则

### 4. 通关界面更新

**通关按钮布局**：
1. **再玩一次** - 重新开始当前关卡
2. **下一关** - 进入下一关（仅在有下一关时显示）
3. **返回菜单** - 返回主菜单

### 5. 游戏界面更新

**游戏UI显示**：
- 左侧：显示当前关卡名称（第一关/第二关）
- 中间：显示当前需要点击的数字
- 右侧：显示进度（已点击/总数）
- 底部：显示剩余时间

### 6. 游戏说明更新

更新了游戏说明，包含关卡信息：
1. 游戏共有两个关卡
2. 第一关：10个图形
3. 第二关：100个图形
4. 每个图形都有一个数字标识
5. 按照数字从小到大的顺序点击图形
6. 点击正确会有高亮反馈并增加时间
7. 点击错误会有震动效果并扣除时间
8. 时间耗尽则游戏失败
9. 按顺序点击完所有数字即可通关

## 技术实现

### 关卡切换逻辑

```javascript
onChangeLevel() {
  this.currentLevel = (this.currentLevel % this.totalLevels) + 1;
  this.buttons[1].text = `关卡: ${this.currentLevel}/${this.totalLevels}`;
  this.buttons[0].text = `开始${this.levelConfig[this.currentLevel].name}`;
}
```

### 下一关逻辑

```javascript
onNextLevel() {
  this.showCompletion = false;
  this.currentLevel++;
  if (this.onNextLevel) {
    this.onNextLevel(this.currentLevel);
  }
}
```

### 关卡启动逻辑

```javascript
startNextLevel(level) {
  this.ui.currentLevel = level;
  this.startGame(this.ui.levelConfig[level].count, level);
}
```

### 进度保存更新

```javascript
const key = `level_${this.gameManager.currentLevel}`;
savedProgress[key] = {
  bestTime: time,
  level: this.gameManager.currentLevel,
  polygonCount: this.gameManager.polygonCount,
  timestamp: Date.now()
};
```

## 游戏流程

### 正常游戏流程

1. **主菜单** → 选择关卡 → 开始游戏
2. **游戏进行** → 按顺序点击数字 → 通关
3. **通关界面** → 选择：
   - 再玩一次（重新开始当前关卡）
   - 下一关（进入下一关，仅第二关通关后不显示）
   - 返回菜单（返回主菜单）

### 关卡切换流程

1. 在主菜单点击"关卡: 1/2"按钮
2. 关卡数字循环切换（1 → 2 → 1 → 2...）
3. "开始游戏"按钮文字相应更新（开始第一关/开始第二关）
